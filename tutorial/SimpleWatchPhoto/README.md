SimpleImageViewer
==============

This sample demonstrates image viewer using the [Tizen.CicularUI](https://github.com/Samsung/Tizen.CircularUI) and [Xamarin.Forms](https://github.com/xamarin/Xamarin.Forms) for Tizen Watch.
