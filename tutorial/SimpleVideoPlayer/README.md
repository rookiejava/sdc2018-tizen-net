SimpleVideoPlayer
==============

This sample demonstrates video playback using the [Tizen.TV.UIControls](https://github.com/Samsung/Tizen.TV.UIControls) and [Xamarin.Forms](https://github.com/xamarin/Xamarin.Forms) for Tizen TV.
